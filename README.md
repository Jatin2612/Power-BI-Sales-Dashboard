# E-Commerce Sales Dashboard (Power BI)

## Project Overview

This project focuses on building an interactive **MISHRA E-Commerce Sales Dashboard** to track and analyze online sales data. The dashboard provides insights into sales performance, customer trends, and product performance using interactive visualizations.

## Features

* Created an interactive dashboard to track and analyze online sales data.
* Implemented complex parameters to enable drill-down analysis within the dashboard.
* Used filters and slicers to allow dynamic and user-driven data exploration.
* Established data connections and joined multiple tables to prepare the dataset.
* Performed calculations and data transformations to support advanced visual analysis.
* Enabled user-driven parameters for customized visualizations.

## Visualizations Used

The dashboard includes multiple customized visualizations such as:

* Bar Chart
* Pie Chart
* Donut Chart
* Clustered Bar Chart
* Scatter Chart
* Line Chart
* Area Chart
* Map Visualization
* Interactive Slicers

## Tools & Technologies

* Power BI
* Data Modeling
* Data Visualization
* DAX (Data Analysis Expressions)

## Insights

The dashboard helps users easily monitor sales performance, identify top-performing products, and analyze trends across different regions and categories.
